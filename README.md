gudb
=====
Welcome to gudb, a prettier gdb interface. 


(Inspired by pudb, the awesome python debugger.)

Requirements
======

1. node.js
2. npm
3. gdb (> v7.3)
4. 
    python2 (>= v2.7, pip install future)
    or
    python3 (>= 3.0)

Usage
======

    git clone https://github.com/typon/gudb 
    cd gudb/build
    npm install
    node main.js <path to program executable>
